Installation of Maven plugin into Eclipse

Let's add m2e plugin to Eclipse (so it can handle maven projects)!
  1. Help > Install New Software...
  2. Select "Indigo - http://download.eclipse.org/releases/indigo" from the drop-down list next to the "Add..." button. Wait until it fetches the repository.
  3. In the checklist view open the "Collaboration" or the "General Purpose Tools" element (either is good).
  4. Check the two m2e elements.
  5. Next, Next, Select "I accept the terms..", Finish.

  6. Wait for the installation procedure, and select "Restart Now" at the end.

Now we have m2e.
