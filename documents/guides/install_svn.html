Installation of SVN

Windows

Install some form of SVN: subversion packages Most people prefer Tortoise client, personally I prefer RapidSVN, because I can easily get a flat view of all related changes in different folders.

Linux

Use Ubuntu Software Center (Ubuntu 11.10+ Oneiric Ocelot) or Synaptics Package Manager in case of pre 11.10 Ubuntu (in case of Fedora, SUSE or other distribution use the package manager of the distribution) to install the following packages:

  • subversion
  • subversion-tools

Personally I prefer to install the RapidSVN GUI client too (available in your distribution's repository).

Mac OS X

Install some form of SVN: subversion packages Personally I prefer RapidSVN GUI client too.
