Installation of Maven

Linux

Use Ubuntu Software Center (Ubuntu 11.10+ Oneiric Ocelot) or Synaptics Package Manager in case of pre 11.10 Ubuntu (in case of Fedora, SUSE or other distribution use the package manager of the distribution) to install the following packages:

  • maven2
  • libmaven-ear-plugin-java
  • libmaven-ejb-plugin-java
  • libmaven-verifier-java
  • libmaven2-core-java
  • libmaven-war-plugin-java

The first package will select a lot more. Installation should set up the environment variables too. Test maven at the command prompt: "mvn --version"

Windows and Mac OS X

Download Maven source and follow it's instructions how to build it. It's important to set up the MAVEN_HOME environment variable, instructions also cover that. On details of how to set an environment variable see the explanation at OPENEMPI_HOME. Test maven at the command prompt: "mvn --version"
