Installation of Java JDK

Linux

Use Ubuntu Software Center (Ubuntu 11.10+ Oneiric Ocelot) or Synaptics Package Manager in case of pre 11.10 Ubuntu (in case of Fedora, SUSE or other distribution use the package manager of the distribution) to install either the openjdk-6-jdk OpenJDK package or the sun-jdk package. Probably OpenJDK is already installed on your system by default so if you want to use SUN JDK, select it from the alternatives (method depends on distribution).

Windows and Mac OS X

Follow Oracle's website how to download and install the JDK.
