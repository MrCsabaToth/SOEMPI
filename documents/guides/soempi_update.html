Updating SOEMPI source

Your source code should be already checked out if you want to update.
  1. Open up a command prompt
  2. Go to the SOEMPI source root (Linux (example): "cd /home/username/SOEMPI/", Windows (example): "cd C:\SOEMPI\openempi")
  3. Say "svn update"
Because you had a version of the source, the SVN client could read any necessary meta data (like repository URL) and perform the operation without any extra parameters.

If you modified something in the source code, please make sure that you don't have any conflicts after the update. If you have any conflicts, you have to resolve it manually.

After source update, refresh your root project in your Eclipse, which will induce a compilation of Eclipse's own compiled version. Also build the soure code from command line and deploy it to JBoss.
