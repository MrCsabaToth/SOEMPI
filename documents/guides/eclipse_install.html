Installation of Eclipse J2EE version

  1. Download "Eclipse IDE for Java EE Developers" (current newest version is 3.7.1, SR1), and unpack it to a preferred location.
    • Note1: Select such a location where you have enough privileges
    • Note2: It's important to get this Java EE Developers version
    • Note3: It's important to select Indigo or newer version, because it has integrated m2e (Maven plugin) support. The older version's m2eclipse Maven plugin has more quirks.
  2. After starting the executable you can accept the default "workspace" workspace name if you already don't have such.
