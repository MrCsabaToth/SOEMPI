Creating database schema

Series of commands (bold) and server responses (regular) and comments (italic) in case of psql prompt. Get to the psql prompt. At the prompt:

  • Connect to the openempi DB with openempi role:
  • postgres=# \c openempi openempi localhost
  • Password for user openempi:
  • psql (8.4.7)
  • SSL connection (cipher: DHE-RSA-AES256-SHA, bits: 256)
  • You are now connected to database "openempi" on host "localhost" as user "openempi".
  • Create the actual schema:
  • postgres=> \i /home/${username}/SOEMPI/data/create_database_schema.sql (or wherever your repository is and this script resides in it)
