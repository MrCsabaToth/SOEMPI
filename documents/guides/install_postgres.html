Installation of PostgreSQL

Windows

Download and install PostgreSQL form it's website. Follow the instructions. The installer includes PgAdmin management utility too.

Linux

Use Ubuntu Software Center (Ubuntu 11.10+ Oneiric Ocelot) or Synaptics Package Manager in case of pre 11.10 Ubuntu (in case of Fedora, SUSE or other distribution use the package manager of the distribution) to install the following packages:

  • postgresql-8.4 or postgresql-9.1 or whichever is the newest supported version on your distribution
  • pgadmin3

Mac OS X

The followings are only our recommendations. If you are an expert with PostgreSQL on Mac OS X, and you have a working system please continue with the schema setup steps.

  • Forget about Fink or MacPorts repositories. Also do not use installer available on PostgreSQL site. Uninstall these if you have them.
  • Create a user "postgres" (any password should do)
  • Download and run PostgreSQL unified installer
