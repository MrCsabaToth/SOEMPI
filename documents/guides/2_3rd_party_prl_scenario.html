Example scenario: multi party setup, 2 3rd parties

Participants:

  • Key Server (KS): provides salts for the Bloom Filter generation.
  • Data Providers (DP): want to match data with other Data Providers.
  • Data Integrator (DI): performs matching on the encoded data coming from DPs.

Steps of the multi-party PRL setup

  1. Install an instance of SOEMPI to the Key Server machine by following the setup guide.
  2. Install an instance of SOEMPI to the Data Integrator machine by following the setup guide.
  3. Install an instance of SOEMPI to the Data Provider 1 machine by following the setup guide.
  4. Install an instance of SOEMPI to the Data Provider 2 machine by following the setup guide.
  5. Generate salts on the Key Server
  6. Initial configure Data Integrator: please perform Key Server settings. (Note: this will be point when a user needs to be created for Data Provider 1 and Data Provider 2 on Key Server and Parameter Manager.)
  7. Configure Data Provider 1: please perform Key Server settings, additionally set the IP addresses of the Parameter Manager and the Data Integrator too.
  8. Configure Data Provider 2: same as with the Data Provider 1.
  9. Import the example clean dataset into Data Provider 1 machine by following the import guide.
  10. Import the example corrupt dataset into Data Provider 2 machine by following the import guide.
  11. Initiate the multi-step PRL link procedure from Data Provider 1 machine by following the guide.
  12. Initiate the multi-step PRL link procedure from Data Provider 2 machine by following the guide.
