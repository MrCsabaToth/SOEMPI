Deleting an existing database schema with pgAdmin III

  1. Login to openempi database if you haven't done so.
  2. Please delete extra tables which are created by your experiments and not part of the standard openempi schema
  3. Open up Databases/openempi/Schemas/public node.
  4. Drop all match pair statistics match pair stat half tables, they are named as tbl_mps_* and tbl_mpsh_*. Right clicking on them and selecting Drop Cascaded option.

  5. Drop all sequences of the previously deleted tables by right clicking on them and selecting Drop Cascaded option.

  6. Drop all link tables, they are named as tbl_lnk_*. Right click on them and select Drop Cascaded option.
  7. Drop all sequences of the previously deleted tables by right clicking on them and selecting Drop Cascaded option.
  8. Delete all rows from column_match_information table.
  9. Delete all rows from person_match.
  10. Delete all rows from person_match_request.
  11. Drop all datset tables, they are named as tbl_ds_*. Right click on them and select Drop Cascaded option.
  12. Drop all sequences of the previously deleted tables by right clicking on them and selecting Drop Cascaded option.
  13. Delete all rows from column_information table.
  14. Delete all rows from dataset table.
