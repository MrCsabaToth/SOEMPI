Getting SOEMPI source

Command line: "svn checkout svn://hiplab.mc.vanderbilt.edu/SOEMPI ~/SOEMPI"

The last parameter is the local folder of the repository, it can be any location you prefer. Pay attention: your user have to have a permission to that place.

You can use the repository location above for GUI SVN clients too.

(Note: currently you need to have permissions on HIPLAB server to be able to checkout. There's no anonymous checkout right now.)
